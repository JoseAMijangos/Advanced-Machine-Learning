Course Description

In this course students learn to use advanced machine learning methods, including dimensionality reduction, support vector machines, ensemble methods, neural nets, and methods for working with time series data.

Course Outcomes

At the end of this class, students should be able to do the following:

  • Apply packages in the Python machine learning ecosystem (which includes NumPy, Scikit Learn, and Pandas) to machine-learning problems of moderate complexity.
  • Apply deep learning to machine-learning problems of moderate complexity, using Google’s TensorFlow library and Python.
  • Explain and apply the methods of support vector machines and ensemble methods (including random forests and boosting).
  • Apply machine machine learning algorithms on time series data sets.
  • Explain the role of optimization in machine learning, and to implement gradient boosting.

Text  Aurelien Géron, Hands-On Machine Learning with Scikit-Learn & TensorFlow, O’Reilly.  Available from Amazon in print or as ebook.  Please respect the author and do not use a pirated PDF of this book.  Optional texts:

  • Cowpertwait and Metcalfe, Introductory Time Series with R, Springer.  (For time-series section of the course.)
  • Rui Miguel Forte, Mastering Predictive Analytics with R, Packt Publishing.  (Nice chapter on time-series data.)
